## 🤝 Contributing to Documentation

We welcome contributions to improve and expand our multilingual documentation:

1. **English**: Base language - all content should be available
2. **Spanish**: High priority for Latin American markets
3. **Catalan**: Regional support for Catalonia

### How to Contribute
1. Fork the repository
2. Create a feature branch for your language
3. Add or update documentation files
4. Follow the established structure and formatting
5. Submit a pull request

### Translation Guidelines
- Maintain technical terminology consistency
- Adapt examples to regional standards when applicable
- Keep the same structure across all languages
- Update the status table above when adding content
